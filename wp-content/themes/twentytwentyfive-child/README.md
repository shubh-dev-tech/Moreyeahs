# Twenty Twenty-Five Child Theme

This is a child theme of Twenty Twenty-Five that preserves all your existing data and custom functionality.

## What's Included

- All ACF field groups and custom blocks
- All REST API endpoints
- All custom functionality from the parent theme
- Your existing data remains intact

## Features Preserved

- Moreyeahs Content Block
- Moreyeahs Service Block
- Stepper Block
- Testimonial Block
- Icon Text Grid
- Image Grid Hover
- Full Width Left Text Section
- Investor Block
- Navigation Next Block
- News Block
- Promo Block
- Purpose Block
- Counter Block
- All REST API endpoints

## How to Use

1. Activate this child theme in WordPress Admin > Appearance > Themes
2. All your existing content and blocks will continue to work
3. Make customizations in this child theme without affecting the parent theme
4. Add custom CSS in `style.css`
5. Add custom PHP functions in `functions.php`

## Benefits

- Safe customizations (parent theme updates won't overwrite your changes)
- All existing data preserved
- All custom blocks continue to work
- Easy to maintain and update

## File Structure

```
twentytwentyfive-child/
├── style.css (child theme styles)
├── functions.php (child theme functions)
├── acf-json/ (ACF field groups)
├── blocks/ (custom blocks)
├── inc/ (includes - ACF blocks, REST API)
└── import-*.php (import scripts)
```
